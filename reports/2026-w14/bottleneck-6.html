特定結果
Organic Search × 新規 の ①→② AC到達率が 17.46%（全体平均 26.10% の -33.1%、-8.64pp）。
月間 723,479セッション（最大ボリューム）。CVR 0.34%は全セグメント中最低水準。SEOランディングページから商品詳細への内部動線が弱く、情報収集意図ユーザーが商品発見前に離脱している。
原因仮説 × 3
なぜ Organic Search × 新規の ①→② AC到達率が低いか？
仮説 1（有力）
SEOランディングページからACページへの内部リンクが弱い
エリアページやブログ記事に着地した新規ユーザーが、個別商品（ACページ）への遷移導線を見つけられない。ファーストビューに商品リストがなくテキスト情報が優勢なため、商品に辿り着く前に離脱する。リピーターは商品URLを直接知っているため影響が小さい。
裏付けデータ:
  • Organic × 新規 ①→②: 17.46% / Organic × リピーター: 36.29%（2倍以上の差）
  • Paid Search × 新規: 31.5%（広告は商品ページ直リンクのため高い）
  • ②→③も31.49%と低く、到達後もカレンダー未到達が続く複合問題
仮説 2
情報収集意図ユーザーが購買行動まで到達しない
「○○旅行 おすすめ」「○○ アクティビティ 種類」等の情報収集キーワードで流入した新規ユーザーは購買意欲が低く、商品詳細を閲覧する前にサイトを離脱する。SEO記事の読了後に購買行動につなぐ導線設計が不足している。
裏付けデータ:
  • CVR 0.34%は全セグメント最低 → 購買意欲の低いユーザーが大量流入
  • 情報収集系クエリはSEO流入の大部分を占める（特に新規ユーザー）
仮説 3
SEOランディングページの速度・UXが低く直帰する
SEO最適化のために作られたエリアページ・ブログ記事のページ速度やモバイルUXが低く、コンテンツを閲覧する前に離脱（直帰）している。ページの読み込み速度やCore Web Vitalsが競合に劣っている可能性がある。
裏付けデータ:
  • SEOランディングページへの直帰率が高い場合、速度・UX問題が存在
  • Mobile 経由の Organic 新規は特に速度影響を受けやすい
打ち手 × 3
Organic Search × 新規の ①→② AC到達率改善施策
施策 A
①→② +5pt
SEOランディングページFVに人気商品カルーセルを配置
エリアページ・ブログ記事のファーストビューに「このエリアの人気アクティビティ TOP5」カルーセルを配置。新規ユーザーがスクロールなしで商品カードをタップでき、直接ACページへ遷移できる導線を作る。
開発仕様 ・SEOランディングページテンプレートにカルーセルコンポーネント追加
・データソース: エリア内の予約数TOP5商品を動的取得
・カード: 画像 + 商品名 + 価格 + 評価（横スクロール、Mobile最適化）
・目標: FV内に商品カードを3枚以上表示
施策 B
①→② +3pt
エリアページに「今すぐ予約できる商品」フィルタービュー追加
エリアページの商品一覧に「今日・明日予約可能」フィルタービューを追加。情報収集ユーザーが購買行動に移るトリガーとして、空き枠のある商品を優先表示し即決を促す。
開発仕様 ・エリアページ商品一覧上部に「今すぐ予約できる」タブ追加
・在庫 API と連携し当日・翌日空き枠あり商品をフィルタリング
・「残り○枠」バッジで緊急性を可視化
・デフォルトは全商品表示のまま、タブ切り替えで即時フィルター
施策 C
①→② +2pt
コンテンツ内への商品内部リンクの自然挿入
SEOブログ記事・エリアページのコンテンツ内に、文脈に合った商品への内部リンクを自然挿入。「○○を体験するなら → 商品名」形式で読者の流れを途切れさせずにACページへ誘導。SEO評価向上とも両立する。
開発仕様 ・CMS で記事テンプレートに「関連商品インライン表示」コンポーネント追加
・商品タグ + エリアタグのマッチングで自動挿入候補を提案
・挿入位置: 記事中盤（読了前の離脱防止）と記事末尾の2箇所
・SEO施策として: 商品ページへの内部リンク強化でランキング改善も期待
検証方法
ABテスト 2週間。対象: Organic Search × 新規ユーザー（全エリアページ・ブログ記事）。
主要指標: ①→② AC到達率（現状 17.46% → 目標 22%以上）。
副次指標: ②→③ カレンダー表示率への影響、直帰率、ページ滞在時間、SEO順位変動。
施策Aから優先実施。Japan（最大エリア）から開始し成功後に全エリア展開。
チームレビュー欄
