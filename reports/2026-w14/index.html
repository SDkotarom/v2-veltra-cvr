# ボトルネック ファネル段階 セグメント 乖離 主要仮説
#2 Desktop × Paid — フォーム完了率の低さ ④→⑤ Desktop / Paid Search -29% フォームの入力項目が多い。Paid経由は比較検討中のユーザーが多く、入力途中の離脱率が高い。
#3 Mobile × Direct — AC到達率の低さ ①→② Mobile / Direct -25% リピーターがトップページから商品に到達できていない。検索UI・お気に入り機能の不足。
#4 Organic × 新規 — AC到達率の低さ ①→② Organic / 新規 -24% SEOランディングページ（エリアページ）から商品への導線が弱い。カテゴリ構造の問題。
#5 Mobile × Organic × Bali — カレンダー表示率 ②→③ Mobile / Organic / Bali -22% #1と同根（速度 + 情報過多）。Baliは商品数が多くACページが特に長い。
#6 Desktop × Organic — フォーム遷移率 ③→④ Desktop / Organic -21% カレンダーは見るが予約に進まない。料金体系が複雑で比較中の離脱。即確定/リクエスト制の表記が分かりにくい。
#7 Mobile × Paid × 新規 — 完了率 ④→⑤ Mobile / Paid / 新規 -19% Mobileフォームの入力UX。新規は会員登録の壁。ゲスト購入への導線不足。
#8 Desktop × Direct × リピーター — カレンダー表示率 ②→③ Desktop / Direct / リピーター -16% リピーターが「前回と同じ商品」を再予約したいが、ACページに過去予約との接続がない。
#9 Mobile × Organic × Europe — AC到達率 ①→② Mobile / Organic / Europe -15% Europeはエリア・都市・商品の階層が深く、カテゴリページから商品に到達するまでのステップが多い。
